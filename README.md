# sacolomina_html5_2526
# Description
This project is a simple HTML5 mini store that allows users to browse and simulate a fake purchase products online. It features a user-friendly interface, a responsive design, and a secure payment system.

# Installation
To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/<span style="color: red">yourusername</span>/sacolomina_html5_2526.git
   ```

2. Navigate to the project directory:
   ```
   cd sacolomina_html5_2526
   ```

3. Install dependencies: (no dependencies are needed)
   ```
   npm install
   ```

4. Start the development server: (not needed, can be skipped)
   ```
   npm start
   ```

# Usage
Once the development server is running, you can access the mini store by opening your web browser and navigating to `http://localhost:3000`.
